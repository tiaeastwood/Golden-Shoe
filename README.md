# Golden Shoe

## Technologies

- React
- Styled Components
- react-router-dom
- json-server

## About

Website redesign demo for Golden Shoe.
Product data comes from a fake API I created using json-server.

## Getting started

1. Install dependencies

```
npm i
```
2. Run the json-server fake API
```
npm run server
```
3. Run the frontend app

```
npm start
```
